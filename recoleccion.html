Sube recolección con seguridad

  • Especialízate: 1–2 recursos para subir maestría rápido.
  • Comida: Pork Pie / carga para más retorno.
  • Rutas: alterna mapas, evita patrones repetidos.
  • Equipo: set de recolección del recurso + botas con sprint + capa defensiva.
  • Encantamientos: prioriza .1/.2/.3 cuando puedas.
Si te siguen, corta hacia diagonal con sprint; evita puentes y chokepoints.

¿Vender bruto o refinar?

Sin focus o sin bono de ciudad, vender bruto ahorra tiempo. Con focus + bono, refina para capturar retorno.

Kit recomendado

  • Herramienta acorde al tier
  • Comida de carga/recursos
  • Montura de carga media
  • Poción de invisibilidad
No recolectes al 100% de peso; deja 20–30% para huir.
